# 🏦 BFSI Analytics Dashboard (SQL + Power BI Project)

## 📌 Project Overview

This project focuses on analyzing Banking, Financial Services, and Insurance (BFSI) data to generate actionable business insights.

The solution integrates SQL for data processing and Power BI for visualization to monitor financial KPIs, customer behavior, and overall business performance.

---

## 🎯 Business Objectives

- Analyze customer financial behavior  
- Track key banking KPIs  
- Identify high-value customers  
- Monitor revenue and transaction trends  
- Support risk and performance analysis  

---

## 🛠️ Tools & Technologies

### 🔹 SQL
- Data extraction and transformation  
- Joins across multiple tables  
- Aggregations and KPI calculations  

### 🔹 Power BI
- Interactive dashboard development  
- KPI visualization  
- Dynamic filters and slicers
